# Apple Frameworks Explorer

An iOS application built with SwiftUI that displays Apple frameworks in a modern grid layout. Users can tap any framework to view detailed information and open the official Apple Developer documentation.

## 🚀 Features

- Built entirely with SwiftUI
- LazyVGrid layout
- Navigation between screens
- Modal sheet presentation
- SafariView integration for external links
- Clean dark UI inspired by Apple design
- Reusable components

## 🧠 Architecture

- MVVM pattern
- Custom Model for Framework data
- Separate Detail View & Web View
- State management using @State and @Binding

## 🛠 Technologies

- Swift
- SwiftUI
- Xcode
- SF Symbols

## 📱 How to Run

1. Clone the repository
2. Open `SwiftUI-Framework.xcodeproj`
3. Run on iOS Simulator
